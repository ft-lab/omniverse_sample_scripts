# DisplayColor

PrimのDisplayColorを取得/設定。  
DisplayColorはジオメトリに対してオブジェクトごとの色を指定できます。  
また、メッシュの場合は面/頂点ごとの色も指定可能です。  
不透明度(Opacity)も指定できます。  

|ファイル|説明|    
|---|---|    
|[GetDisplayColor.py](./GetDisplayColor.py)|選択PrimのDisplayColorを取得|  
|[ClearDisplayColor.py](./ClearDisplayColor.py)|選択PrimのDisplayColor/DisplayColorOpacityを削除|  
|[CreateConstantDisplayColor.py](./CreateConstantDisplayColor.py)|選択PrimにDisplayColorを指定|  
|[CreateUniformDisplayColor.py](./CreateUniformDisplayColor.py)|選択Primがメッシュの場合に面ごとにDisplayColorを指定|  
|[CreateVertexDisplayColor.py](./CreateVertexDisplayColor.py)|選択Primがメッシュの場合に頂点ごとにDisplayColorを指定|
